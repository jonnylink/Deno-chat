# ChatGPT Wrapper

A super simple Node.js wrapper for ChatGPT using OpenAI's API.

## Setup

1. Clone the repository.
2. Run `yarn install`.
3. Create a `.env` file with your OpenAI API key.
4. Run `yarn start`
5. Send a POST request to `/chat` with a JSON body:
```json
{"message": "Your message here"}
```
